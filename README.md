## Сборка
Собрать из исходного кода можно так:
1. Клонировать репозиторий:
```sh
git clone https://github.com/victorbaldin56/triangles3D.git
```
2. Сгенерировать конфигурацию сборки:
```sh
cmake -S . -B build
```
3. Вызвать систему сборки:
```sh
cmake --build build
```
